## How to Upgrade

Follow these steps to update your haha Documentation instance to the latest version.

<Steps>
  <Step title="Check Current Version" icon="package">
    Run the version check command:

    <CodeGroup tabs="CLI,npm">
      ```bash
      haha-docs version
      ```
      ```bash
      npm list haha-docs
      ```
    </CodeGroup>
  </Step>
  <Step title="Backup Configuration" icon="database">
    Export your settings:

    ```bash
    haha-docs export-config > backup.json
    ```
  </Step>
  <Step title="Install Update" icon="download">
    Update via package manager:

    <Tabs>
      <Tab title="npm" icon="package">
        ```bash
        npm install haha-docs@latest
        ```
      </Tab>
      <Tab title="Docker" icon="docker">
        ```bash
        docker pull haha-docs:latest
        docker run -p 3000:3000 haha-docs:latest
        ```
      </Tab>
    </Tabs>
  </Step>
  <Step title="Verify and Restart" icon="check-circle">
    Restart your server and test key features.
  </Step>
</Steps>
